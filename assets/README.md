# Assets

This directory stores curated, presentation-ready assets that should remain under version control.

## Policy

- Keep raw experiment dumps in `outputs/quantum_oscillator/artifacts/`
- Keep non-figure analysis files in `outputs/quantum_oscillator/analysis/`
- Keep generated figures in `results/quantum_oscillator/`
- Keep only the most meaningful, report-ready figures in `assets/figures/`

The goal is to make the repository landing page and the final report easy to read without versioning every intermediate artifact.
